\# ESP8266 WiFi RSSI FSM



This project implements a Finite State Machine (FSM) on ESP8266 to scan nearby WiFi networks and measure RSSI values.



\## Features

\- WiFi network scanning

\- RSSI measurement (dBm)

\- Serial Monitor output

\- Structured FSM-based firmware



\## Folder Structure

\- src/ → Arduino source code

\- docs/ → Output logs / screenshots
